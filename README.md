🎓 Task 4 – Student Performance Dashboard (Streamlit Project)

📌 Overview
This project is a Data Science + Streamlit-based interactive dashboard that analyzes and visualizes student performance data. It helps understand key academic factors affecting student outcomes using data visualization and basic analytics.

The project is deployed using Streamlit Cloud and demonstrated through a YouTube video.

🚀 Live Demo (Streamlit App)
👉 https://synent-task4-app-dashboard-smitkumar-app.streamlit.app/

🎥 Project Demo Video
👉 https://youtu.be/PB90HfgzCqs

📊 Dataset
Student Performance Dataset

Features:
- Study Hours
- Previous Scores
- Attendance
- Sleep Hours
- Practice Papers Solved

Target Variable:
- Final Performance Score
